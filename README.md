# Rpi Board Test 

## Spec 

- Raspberry Pi Compute Module 3+ 
- [Raspbian Buster Lite](https://www.raspberrypi.org/downloads/raspbian/)


## Requirement 
- [Node.js](https://nodejs.org/ko/)
- Npm or Yarn (Package manager)


# Usage 
## Npm
### init 
```bash
npm install 
```

### run  
```bash
npm start 
```


## Yarn 
### init 
```bash
npm install 
```

### run  
```bash
npm start 
```
